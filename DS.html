Stack

An abstract data type that serves as a collection of elements which utilizes the first-in last-out (FILO) method. It consist of three main operations; Push: adds an element to the collection, Pop: removes the most recently added element that was not yet removed, and peek: returns the value at the last element.



LinkedList

In computer science, a linked list is a linear collection of data elements whose order is not given by their physical placement in memory. Instead, each element points to the next. It is a data structure consisting of a collection of nodes which together represent a sequence.
